My Radar

Posted on 2023-10-01

My Radar is a anti-theft device that uses an ultrasonic sensor to detect objects and alert the user via a mobile app. It uses a simple servo motor to monitor around the surrounding area. It is designed to be used in conjunction with a bicycle lock, providing an additional layer of security. The device is powered by a ESP32 Dev Module and communicates with a mobile app an access WiFi point.

The My Radar project is a great example of how technology can be used to enhance security and provide peace of mind for users. It is a simple yet effective solution that can help prevent bike theft and protect valuable property.

The My Radar project is an open-source project, and the code is available on GitHub. The project is designed to be easy to use and modify, allowing users to customize the device to their specific needs. The project also includes a mobile app that is available for both iOS and Android devices.
